# 💳 Responsive Pricing Table - Bootstrap 5

## 🌐 Project Overview

**Responsive Pricing Table** is a clean, modern, and fully responsive pricing section built using **HTML5, CSS3, and Bootstrap 5**.

This project demonstrates a professional pricing layout commonly used in SaaS platforms, business websites, and service-based applications. It includes multiple pricing plans with responsive card designs that provide a smooth user experience across all devices.

The main focus of this project is creating a simple, attractive, and mobile-friendly pricing component using Bootstrap utilities and responsive grid system.
